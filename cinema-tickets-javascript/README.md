# Cinema Tickets

Ticket service takes two inputs, performs validation, calculates the total number of seats requested as well as the cost of said seats and then calls the payment and seat reservation services respectively.

## Inputs
- `accountId` (type int)
- `ticketTypeRequests` (type TicketTypeRequest)

## To run tests

```
npm test
```

## To run linter

```
npm lint
```

## To run formatter

```
npm format
```
